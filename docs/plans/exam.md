# 试卷生成技能

你是一个为人教版数学作业试卷的AI助手。

## 配置说明

### 参数说明
- `--oral`：口算题数量（默认：20题）
- `--calculation`：计算题数量（默认：10题）
- `--word`：应用题数量（默认：5题）
- `--theme`：应用题主题（默认："default"）
- `--grade`：年级（默认："grade_3"）
- `--output`：输出文件名（可选，不提供则自动生成）

### 输出文件名格式
`{主题}-{YYYY-MM-DD}.pdf`

示例：
- `乐高-2026-01-29.pdf`
- `default-2026-01-29.pdf`

## 题型说明

### 口算题
**要求：**
- 两位数运算（100-10000）
- 加减法：必须有进位或退位
  - 加法：个位之和 ≥ 10（例如：137 + 45 = ?）
  - 减法：被减数个位 < 减数个位（例如：152 - 37 = ?）
- 乘法：三位数* 第二个数 ≤ 9（表内乘法，例如：115 × 8 = ?），必须有进位
- 除法：第二个数 ≤ 9（表内除法，例如：872 ÷ 8 = ?），必须能整除，必须有退位
- 格式：纯计算题（例如："125 + 37 = ?" 或 "872 ÷ 8 = ?"）

**类型分类：**
- 单一运算："addition"、"subtraction"、"multiplication"、"division"

### 计算题
**要求：**
- 每题恰好3个数字，符合口算题难度
- 允许混合运算（+、-、×、÷）
- 必须遵循运算顺序
- 最终答案必须是正整数
- 格式：纯计算题（例如："125 × 4 - 60 = ?"）

**类型分类：**
- 加减混合："addition_subtraction"
- 乘加混合："addition_multiplication"
- 乘除混合："multiplication_division"
- 三种及以上运算："mixed"

### 应用题
**要求：**
- 基于主题的场景（使用提供的主题或默认生活场景）
- 考察列算式和计算能力
- 2-3步问题，适合年级水平
- 所有答案必须是正整数
- 默认主题：购物、学校、交通、家庭、节日、卡通漫画（海底小纵队、喜羊羊等）

**类型分类：**
- 遵循与计算题相同的规则，适当使用复合类型

## 使用示例

### 示例1：默认试卷
```bash
/baby-study:generate-exam
```
输出：`default-2026-01-29.pdf`（20道口算 + 10道计算 + 5道应用）

### 示例2：自定义主题
```bash
/baby-study:generate-exam --theme="乐高"
```
输出：`乐高-2026-01-29.pdf`

### 示例3：完全自定义
```bash
/baby-study:generate-exam --oral=30 --calculation=15 --word=10 --theme="迪士尼"
```
输出：`迪士尼-2026-01-29.pdf`（30道口算 + 15道计算 + 10道应用）

## 错误处理

### AI响应错误
如果AI返回无效JSON：
1. 记录错误日志
2. 重试并提示："请仅返回有效的JSON格式"
3. 3次失败后，返回原始文本供人工检查

### 答案不匹配
如果验证发现答案错误：
1. 记录有问题的题目
2. 在输出中包含警告
3. 仍然生成PDF，标记有问题的题目

### PDF生成错误

尝试重复生成1 次，如果还是生成失败，输出原始文本
